# Security Policy

Thank you for helping keep SuiOutKit secure.

Reporting
- Please report security vulnerabilities privately to the maintainers so we can triage and remediate them before public disclosure.
- Preferred channels:
  - Open a private GitHub Security Advisory for this repository (recommended), or
  - Email: [the3rdweblabs@gmail.com](mailto:the3rdweblabs@gmail.com).

What to include
- A clear description of the issue and steps to reproduce (minimal PoC if available).
- The affected component (backend, SDK, contracts) and version/commit.
- Any relevant logs, request IDs, or sample payloads (do NOT include private keys or full secrets).

Handling and timeline
- We aim to acknowledge reports within 72 hours and provide a remediation timeline or mitigations.
- For reports requiring immediate action (active exploitation), please mark the report as high priority in the advisory or subject line.

Disclosure policy
- We prefer coordinated disclosure. Do not publicly post vulnerable code, secrets, or exploit details until a fix is available or mutually agreed timelines expire.

Thanks for helping keep the project safe.
